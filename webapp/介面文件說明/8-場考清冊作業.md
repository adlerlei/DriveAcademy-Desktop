# 開發文件：場考清冊作業

## 1. 總覽

此功能用於安排學員的場地考照（路試），並製作對應的清冊與計分表。操作人員可以查詢已在結訓名冊中的學員，為其設定路試日期、組別與考試項目，系統會自動編排考試號碼，並可最終列印出多種報表。

## 2. 功能區塊說明

### A. 學員查詢區

說明：此區塊的欄位用於查詢學員資料。輸入任一欄位資訊後，系統會自動查找並帶入學員的詳細資料。

- **欄位：學員編號**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

- **欄位：學員姓名**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

- **欄位：身分證號碼**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

### B. 學員資料顯示區（唯讀）

說明：查詢到學員後，其相關資料會顯示在此，此區塊的欄位為唯讀，不可編輯。

- 名冊號碼
- 出生日期
- 訓練班別
- 期別
- 梯次

資料來源：所有資料皆來自「學員資料作業」中已建檔的學員資料庫。

### C. 場考登錄區（可編輯/自動生成）

- **欄位：路試日期**
  - UI 元件：文字輸入框或日期選擇器
  - 說明：設定學員的路考日期
  - 資料來源：使用者輸入

- **欄位：組別**
  - UI 元件：文字輸入框
  - 說明：設定學員被分配到的考試組別
  - 資料來源：使用者輸入

- **欄位：號碼**
  - UI 元件：文字輸入框
  - 說明：此欄位為自動生成的流水號，代表學員的考試號碼，使用者不需填寫
  - 資料來源：系統自動生成（每新增一位學員即遞增）

- **欄位：路考項目**
  - UI 元件：下拉選單
  - 說明：設定學員的路考項目
  - 固定資料：
    - '1'
    - '2'
    - '3'

### D. 列印紀錄區

說明：此區塊用於篩選並列印特定範圍的「場考計分表」。

- **欄位：**
  - **時段**：下拉選單（上午/下午）
  - **開始號碼**：文字輸入框
  - **結束號碼**：文字輸入框

## 3. 功能按鈕

- **新增場考清冊**
  - 功能：將目前設定的場考資訊更新至學員資料庫，並將該學員加入下方的場考清冊中
  - 前置條件：必須先查詢到一位學員

- **列印場考清冊（監理站用）**
  - 功能：產生一份符合監理站報備格式的官方場考清冊報表並觸發列印

- **列印場考清冊（駕訓班用）**
  - 功能：產生一份供駕訓班內部使用的場考清冊報表並觸發列印

- **匯出 場考清冊 文件**
  - 功能：將下方場考清冊中的所有學員資料，匯出成外部文件

- **列印紀錄**
  - 功能：根據「列印紀錄區」設定的條件（時段、號碼範圍），篩選清單中的學員並列印出場考計分表

## 4. 場考清冊

- **UI 元件**：表格（Data Grid / Table）
- **說明**：用於暫存本次操作中，所有已加入場考清冊的學員
- **列表欄位**：
  - 號碼
  - 學員編號
  - 名冊號碼
  - 梯次
  - 學員姓名
  - 出生日期
  - 身分證號碼
  - 路試日期
  - 組別
  - 路考項目
