# 開發文件：教練資料作業

## 1. 總覽

此功能用於建立、查詢、修改與刪除駕訓班的教練基本資料。

## 2. 輸入/查詢欄位說明

### 欄位：教練編號
- **UI 元件**：文字輸入框
- **說明**：教練的唯一識別編號。此欄位也可用於查詢，輸入後會自動帶入該教練的所有資料。
- **資料來源**：使用者輸入

### 欄位：教練姓名
- **UI 元件**：文字輸入框
- **資料來源**：使用者輸入

### 欄位：出生日期
- **UI 元件**：文字輸入框或日期選擇器
- **資料來源**：使用者輸入

### 欄位：教練證號
- **UI 元件**：文字輸入框
- **說明**：教練的官方證照號碼
- **資料來源**：使用者輸入

## 3. 功能按鈕

- **新增**
  - 功能：將目前在欄位中輸入的資料，作為一筆新的教練紀錄存入資料庫
  - 前置條件：若已透過查詢帶出資料，則無法新增，需使用「修改」功能

- **修改**
  - 功能：更新目前查詢帶出的教練資料
  - 前置條件：必須先透過「教練編號」查詢到一位教練

- **刪除**
  - 功能：刪除目前查詢帶出的教練資料
  - 前置條件：必須先透過「教練編號」查詢到一位教練

## 4. 資料列表

- **UI 元件**：表格（Data Grid / Table）
- **說明**：用於顯示資料庫中所有已建檔的教練紀錄。新增、修改或刪除資料後，列表會自動刷新。
- **列表欄位**：
  - 教練編號
  - 姓名
  - 出生日期
  - 教練證號
