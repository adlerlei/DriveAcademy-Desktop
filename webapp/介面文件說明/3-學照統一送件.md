# 開發文件：學照統一送件

## 1. 總覽

此功能用於批次處理學員的學習駕照（學照）送件作業。操作人員可以透過查詢找到特定學員，設定統一的「送件日期」，並將其加入下方的送件清單中，最後可將清單匯出成文件。

## 2. 功能區塊說明

### A. 學員查詢區

說明：此區塊的欄位主要用於查詢學員資料。輸入任一欄位的資訊後，系統會自動從學員資料庫中查找對應的學員，並將其詳細資料帶入下方相關欄位中。

- **欄位：學員編號**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

- **欄位：學員姓名**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

- **欄位：身分證號**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

- **欄位：聯絡電話**
  - UI 元件：文字輸入框
  - 資料來源：使用者輸入（用於查詢）

### B. 學員資料顯示區（唯讀）

說明：當查詢到學員後，其相關資料會顯示在此區塊，此區塊的欄位為唯讀，不可編輯。

- 出生日期
- 學員信箱
- 備註
- 戶籍地址（包含郵遞區號、縣市、詳細地址）

資料來源：所有資料皆來自「學員資料作業」中已建檔的學員資料庫。

### C. 送件作業區

- **欄位：送件日期**
  - UI 元件：日期選擇器或文字輸入框
  - 說明：設定本次要送件的日期。此為必填欄位。
  - 資料來源：使用者輸入

## 3. 功能按鈕

- **學照統一送件**
  - 功能：將目前查詢到的學員，連同已設定的「送件日期」，加入到下方的送件清單中，並將送件日期更新回該學員的資料庫紀錄中。
  - 前置條件：必須先查詢到一位學員，且「送件日期」欄位不可為空。

- **匯出文件**
  - 功能：將下方送件清單中的所有學員資料，匯出成外部文件（如 CSV 或 Excel）。

## 4. 送件清單

- **UI 元件**：表格（Data Grid / Table）
- **說明**：用於暫存本次操作中，所有已加入送件流程的學員。每次點擊「學照統一送件」按鈕，就會將一位學員新增至此清單。
- **列表欄位**：
  - 身分證號
  - 出生日期
  - 學員姓名
  - 區號
  - 縣市區
  - 戶籍地址
  - 手機
  - E-mail
