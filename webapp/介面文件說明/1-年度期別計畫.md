# 開發文件：年度期別計畫

## 1. 總覽

此功能用於建立駕訓班的年度開班計畫，包含設定各班別的期別、梯次與開/結訓日期。

## 2. 輸入欄位說明

### 訓練班別
- **UI 元件**：2個連動的下拉選單
- **說明**：選擇班別代碼會自動帶入對應的名稱，反之亦然。必填。
- **固定資料**：
  - `1`：普通小型車班
  - `2`：大貨車班
  - `3`：大客車班
  - `4`：聯結車班
  - `5`：職業小型車班
  - `6`：普通重機車班
  - `7`：大型重機車班
  - `8`：小型車逕升大客車班

### 梯次
- **UI 元件**：下拉選單
- **說明**：選擇班級的梯次。必填。
- **固定資料**：
  - `A`
  - `B`

### 期別
- **UI 元件**：文字輸入框
- **說明**：使用者自行輸入期別編號。必填。
- **資料來源**：使用者輸入

### 年度
- **UI 元件**：文字輸入框
- **說明**：使用者自行輸入民國年。必填。
- **資料來源**：使用者輸入

### 上課期別代碼
- **UI 元件**：文字輸入框（自動生成，無須輸入）
- **說明**：由系統根據其他欄位自動生成。規則：訓練班別代碼 + 0 + 期別 + 梯次。
- **資料來源**：自動生成

### 開訓日期
- **UI 元件**：文字輸入框或日期選擇器
- **說明**：使用者選擇或輸入課程開始日期。必填。
- **資料來源**：使用者輸入

### 結訓日期
- **UI 元件**：文字輸入框或日期選擇器
- **說明**：使用者選擇或輸入課程結束日期。必填。
- **資料來源**：使用者輸入

## 3. 功能按鈕

- **新增**：驗證所有必填欄位後，將整筆資料存入資料庫。
- **刪除**：從資料庫中刪除在列表中選定的一或多筆項目。
- **匯出文件**：將列表中選定的一或多筆項目匯出成檔案。

## 4. 資料列表

- **UI 元件**：表格（Data Grid / Table）
- **說明**：顯示所有已儲存的年度期別計畫
- **資料來源**：從資料庫抓取
- **列表欄位**：
  - 訓練班別名稱
  - 年度
  - 期別編號
  - 開訓日期
  - 結訓日期
  - 上課期別代碼
